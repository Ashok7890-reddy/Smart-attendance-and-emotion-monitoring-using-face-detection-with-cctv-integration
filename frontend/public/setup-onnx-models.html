🚀 Benefits of ONNX Models

  • 🎯 Higher Accuracy: 95-99% recognition accuracy vs 85-90% with face-api.js
  • 🔬 Better Embeddings: 512-dimensional ArcFace embeddings vs 128D FaceNet
  • 📐 Precise Detection: YuNet provides more accurate face bounding boxes
  • 🏷️ Landmark Detection: 5-point facial landmarks for better alignment
  • 🛡️ Robust Recognition: Better performance in varying lighting conditions
  • ⚡ Optimized Performance: WASM-optimized for browser execution
  • 🔄 Intelligent Fallback: Automatically falls back to face-api.js if needed

📋 Manual Setup Instructions

If automatic download fails, follow these steps:

  1. Create directory: frontend/public/models/onnx/
  2. Create directory: frontend/public/onnx-wasm/
  3. Download YuNet model from OpenCV Zoo
  4. Download ArcFace model from ONNX Model Zoo
  5. Copy ONNX Runtime WASM files
