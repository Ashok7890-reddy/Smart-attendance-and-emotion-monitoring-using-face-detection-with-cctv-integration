🔍 Troubleshooting

If TinyFaceDetector Not Working

Symptoms: Always falling back to SSD MobileNet
Solutions:
• Check model files exist in /models/
• Improve image quality (better lighting)
• Clear browser cache (Ctrl+Shift+R)
• Check console for loading errors

If Performance Not Improved

Symptoms: Detection times still >200ms
Solutions:
• Verify TinyFaceDetector is being used (check console)
• Test with good quality images
• Monitor efficiency percentage
• Use performance monitor tool
